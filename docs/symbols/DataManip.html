  • <static> {Array} DataManip.filter(records, name, value)
    Filters records based on the given field name and value. A record must both have a field with the given name and that fields value must match the given value to be selected.The data structure does not need to be consistent (records can have missing or extra fields). 
    Example
var data = [
  { id: 1, name: 'Bob', score: 92 },
  { id: 2, name: 'Sam', score: 68 },
  { id: 3, name: 'Bob', score: 89 },
  { id: 4, name: 'Sam', score: 72 },
  { id: 5, name: 'Sam', score: 81 }
];

DataManip.filter( data, 'name', 'Bob' );
>>>
[
  { id: 1, name: 'Bob', score: 92 },
  { id: 3, name: 'Bob', score: 89 }
];
    Parameters:
    {Array} records
    records to act on
    {String} name
    of the 'seeking' field
    value
    seeking field value
    Returns:
    {Array} filtered set of items
  • <static> {Hash} DataManip.pivot(records, field, flat)
    Pivots records on the given field (member). The structure of the result is controlled by value of the 'flat' argument. 
    Example
var data = [
  { id: 1, name: 'Bob', score: 92 },
  { id: 2, name: 'Sam', score: 68 },
  { id: 3, name: 'Bob', score: 89 },
  { id: 4, name: 'Sam', score: 72 },
  { id: 5, name: 'Sam', score: 81 }
];
 
DataManip.pivot( data, 'name' );
>>>
{
  Bob: [
    { id: 1, name: 'Bob', score: 92 },
    { id: 3, name: 'Bob', score: 89 }
  ],
  Sam: [
    { id: 2, name: 'Sam', score: 68 },
    { id: 4, name: 'Sam', score: 72 },
    { id: 5, name: 'Sam', score: 81 }
  ]
]
    Parameters:
    {Array} records
    records to act on
    {String} field
    member name to pivot on
    {boolean} flat Optional, Default: false
    controls result structure
    true
    map of values is returned (value selected is the 'last' one matching)
    false
    (default) map of arrays of values is returned.
    Returns:
    {Hash} pivoted view of records
